LatinMorphologicalForma and FormUrns

Implementations of the LatinMorphologicalForm are interchangeable with FormUrns in the Tabulae collection.

We can directly construct LatinMorphologicalForms with type-specific parameters. For example, a LMFFiniteVerb can be directly constructed with values for the person, number, tense, mood and voice, and a LatinNoun with values for gender, case, and number.

using Tabulae
verb = LMFFiniteVerb(3,1,1,1,1)
formurn(verb)
noun = LatinNoun(1,1,1)

Tabulae.jl can represent LatinMorphologicalForms as FormUrns belonging to a collection of URN values for Latin morphology. We can convert any LatinMorphologicalForm to a FormUrn with the formurn function.

nounUrn = formurn(noun)

Object identifiers in this collection are ten-character strings with each character representing an integer code for the following morphological properties:

  1. "part of speech" (analytical type)
  2. person
  3. number
  4. tense
  5. mood
  6. voice
  7. gender
  8. case
  9. degree
  10. uninflected category

Examples

Tabulae.jl offers functions for extracting specific integer values from a FormUrn, and for composing labels for the values.

using Tabulae # hide
noun = LatinNoun(1,1,1) # hide
urn = formurn(noun)
Tabulae.poslabel(urn)
Tabulae.genderlabel(urn)
Tabulae.caselabel(urn)
Tabulae.numberlabel(urn)
Tabulae.personlabel(urn)

Roundtripping
